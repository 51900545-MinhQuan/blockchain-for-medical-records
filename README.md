# blockchain-for-medical-records

Chạy web:
1. npm install
2. npm start

rimraf node_modules (clear node_modules trước khi up lên git)

Thay đổi nhỏ cho web trước khi bắt đầu nghiên cứu đề tài chính:
1. Đổi logo của web sang logo bệnh viện
2. Tạo email phụ để send email cho chức năng quên mật khẩu
3. Thay đổi thông tin cá nhân, ảnh đại diện cho người dùng
4. Thêm role cho bác sĩ
